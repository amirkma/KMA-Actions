# 🚀 KMA DOWNLOADER PRO

دانلودر قدرتمند با قابلیت دور زدن خطای 410 و پشتیبانی از محتوای بزرگسالان

## ✨ ویژگی‌ها

- ✅ دانلود از هر سایتی (YouTube, Twitter, Instagram, و...)
- 🛡️ **دور زدن خطای HTTP 410** با ۵ روش مختلف
- 🔞 پشتیبانی از محتوای بزرگسالان (با محافظت خودکار)
- 📦 اسپلیت خودکار فایل‌های بزرگ (قابل تنظیم)
- 🌍 پشتیبانی از پروکسی و VPN
- 🔄 Retry هوشمند (تا ۵ روش مختلف)
- 📊 گزارش کامل از وضعیت دانلود

## 📖 نحوه استفاده

1. به بخش **Actions** بروید
2. روی **KMA Downloader Pro** کلیک کنید
3. دکمه **Run workflow** را بزنید
4. اطلاعات زیر رو وارد کنید:
   - لینک ویدیو
   - کیفیت (best/720p/1080p)
   - آیا محتوای بزرگسالانه؟ (بله/خیر)
   - استفاده از پروکسی (برای دور زدن 410)
5. صبر کنید تا دانلود انجام بشه
6. فایل رو از پوشه `downloads` دانلود کنید

## 🛠️ رفع خطای 410

اگر با خطای 410 مواجه شدید:
1. گزینه **Use Proxy** رو فعال کنید
2. لینک رو با `scripts/check_url.sh` چک کنید
3. از کیفیت پایین‌تر استفاده کنید
4. لینک رو تغییر بدید

## ⚠️ نکات مهم

- ریپازیتوری رو **PRIVATE** کنید
- حجم هر فایل نهایت 2GB (GitHub limit)
- برای فایل‌های بزرگتر از 95MB خودکار split میشه
- محتوای بزرگسالان در پوشه `adult_downloads` ذخیره میشه

## 📞 پشتیبانی

اگه بازم ارور 410 داد، لینک رو برام بفرست تا بررسی کنم.
